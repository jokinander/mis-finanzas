# 💰 Mis Finanzas

App personal de finanzas con cotización del dólar en tiempo real.

## Features

- 📊 Seguimiento de ingresos y egresos en **pesos y dólares**
- 💵 Cotización del dólar en tiempo real (Blue, Oficial, MEP) via [DolarAPI.com](https://dolarapi.com)
- 📈 Balance dinámico mes a mes
- 🏦 Patrimonio separado en ARS y USD
- 🔍 Análisis y recomendaciones automáticas
- 📱 Responsive y dark mode

## Setup

```bash
npm install
npm run dev
```

## Deploy en Vercel

1. Subir a GitHub
2. Conectar el repo en [vercel.com](https://vercel.com)
3. Framework: **Vite**
4. Deploy automático

## API

Usa [DolarAPI.com](https://dolarapi.com) - gratuita, sin API key, se actualiza cada 5 minutos.
